管理学院服务器R-servers登录方法_通用

Created2023-06-07|Updated2023-06-07|团队协作
|Post View:

常规登录

服务器地址为10.9.65.31
R-servers端口为8787
在浏览器输入10.9.65.31:8787进入登录界面
R-servers账号密码与linux ssh登录账号密码相同

注意事项

  1. 学校提供的easyconnect VPN无法登录,因为VPN仅开放22端口,无8787端口访问权限
  2. 如无特殊情况,新建文件夹以上传文件,避免上传至R文件夹
  3. 单独账户一般仅拥有本账户目录权限,地址为”~/“而无”/“访问使用权限

使用配置文件自动选择默认镜像

首先使用 cat ~/.Rprofile 查看全局配置文件是否存在, 如不存在可以使用 touch ~/.Rprofile 创建全局配置文件.

使用 vim ~/.Rprofile 对全局配置文件进行编辑, 输入 i 进入编辑模式, 使用 Esc 退出编辑模式, 使用 :wq! 或 shift + zz (即 ZZ )退出编辑.

注意, R会自动读取工作目录下的.Rprofile文件进行配置, 其实就当于source .Rprofile, 所以当工作目录下自带了一个新的.Rprofile文件, 便不会读取用户根目录下的.Rprofile文件.
