Ubuntu 修改用户的默认目录

Ubuntu 系统中，每次打开终端，就会进入系统默认的用户目录/home/username，有时候我们想要修改用户默认目录，以更快速地进入项目进行操作，这时候我们就需要通过以下方法来修改用户默认目录。

第一步
执行以下代码，修改passwd配置文件
sudo vi /etc/passwd

第二步
找到用户所在行，vim是用 :/你的用户名 来快速查找，找到所在行类似下面这种：

将其中的/home/smy改成你想要修改成的目录（采用绝对路径），如/dev/sda1/home/smy，其余内容都别改，保存后关闭。

第三步
关闭终端，重新打开，就自动进入你修改的目录了。

第四步

将原根目录的.bashrc和.project文件复制入新根目录.

免密上传

可生成并上传本地密钥到服务器, 完成免密上传.
